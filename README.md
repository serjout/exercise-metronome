# Exercise Metronome

Русскоязычный метроном для упражнения:

- настройка BPM;
- настройка количества тактов;
- зеленая команда `сжимаем` на тактах 1...N;
- синяя команда `глотаем` последней командой на N+1;
- таймер упражнения с паузой по пробелу.

Открывается как статическая GitHub Pages-страница или SPA/PWA-приложение.
После первого открытия по HTTPS service worker кэширует оболочку приложения,
поэтому метроном продолжает работать оффлайн. При прямом открытии `index.html`
из файла приложение тоже остается самодостаточным.
